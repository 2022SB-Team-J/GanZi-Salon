## 1. 소개
**꼭 잘라봐야만 아나요?**

미용실에 가지 않고도 헤어스타일이 어떻게 변화할 지 아는 방법 아시는 분?

GanZi-Salon 은 여러분에게 솔루션을 제시합니다. 원하는 헤어스타일을 내 셀카와 함께 합성한 결과를 사용자에게 제공합니다. 결과물을 비교하면서 마음에 드는 헤어스타일을 찾아보세요!
